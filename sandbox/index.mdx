# Heify Sandbox

The **Heify Sandbox** is your visual command center for managing and creating transcriptions and configurations, and analyzing results—all without writing a single line of code.

<Info>
  **No code required**: The Sandbox provides a complete browser-based interface to use Heify's full capabilities before integrating with your application.
</Info>

## Key Features

### <Icon icon="gear" iconType="regular" /> Visual Configuration Builder

Design complex transcription configurations using our intuitive interface:

- **Drag-and-drop** custom vocabulary
- **One-click templates** for common extraction fields (sentiment, classification, etc.)
- **Quick actions** for predefined field types

### <Icon icon="microphone" iconType="regular" /> Transcribe


- **Upload files** or provide URLs to start transcriptions
- **Process multiple files at once**

### <Icon icon="chart-line" iconType="regular" /> Advanced Analytics

Gain insights from your transcription data:

- **Distribution charts** for speakers, languages, and durations
- **Trend analysis** over time
- **Correlation heatmaps** for numerical fields
- **Co-occurrence matrices** for array data
- **AI-generated summaries** of key findings
- **AI-generated insights** of key findings
- **Automated anomaly detection**
- **Export data** for external analysis

*And much more...*

### <Icon icon="list-check" iconType="regular" /> Transcriptions Management

Organize your transcriptions:

- Mark transcriptions for review
- Track progress through stages
- Filter and search across thousands of files
- Export data for external analysis

<Steps>
  <Step title="Access the Sandbox">

  </Step>
  
  <Step title="Create Your First Configuration">

    [Learn more →](/sandbox/configurations)
