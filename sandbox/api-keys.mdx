## Configure Sandbox
<Info>
  To use the Sandbox, you need to create an API key and configure the Sandbox with that key.
</Info>    
    
    **How to configure:**
    1. Create a new API key and copy it
    2. Click on "Configure" button
    2. Click the "Test Connection" button
    3. Wait for the system to ping Heify servers
    
    <Warning>
      If the test fails, **wait a few seconds and try again**. If it still fails, check your internet connection and browser settings (ad blockers, VPNs, or firewalls may interfere) and try again.
    </Warning>
