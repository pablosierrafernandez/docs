# Quickstart Guide

This guide will walk you through making your first transcription request with the Heify API. In just a few minutes, you'll be able to transcribe audio files and retrieve results.

## Prerequisites

Before you begin, you'll need:

- **An audio or video file** to transcribe (see [supported formats](/index#supported-media-formats))
- **A Heify API key** ([manage keys here](/sandbox/api-keys))
- **A configuration ID** (you can create one in the Sandbox or via the API)

<CodeGroup>

```bash cURL
curl -X POST https://api.heify.com/create-configuration \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "tag": "My First Configuration",
    "summary": true,
    "summary_language": "en"
  }'
```

```python Python
import requests

url = "https://api.heify.com/create-configuration"
headers = {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY"
}
payload = {
    "tag": "My First Configuration",
    "summary": True,
    "summary_language": "en"
}

response = requests.post(url, headers=headers, json=payload)
data = response.json()
configuration_id = data["data"]["configuration_id"]
print(f"Configuration created: {configuration_id}")
```

</CodeGroup>

**Response:**

```json
{
  "data": {
    "message": "Configuration created successfully",
    "configuration_id": "a1b2c3d4..."
  }
}
```

<Note>
  Save the `configuration_id` - you'll need it for the next step!
</Note>

## Step 2: Submit Audio for Transcription

Now submit an audio file using one of two methods:

### Option A: From a Public URL

If your audio is already hosted online:

<CodeGroup>

```bash cURL
curl -X POST https://api.heify.com/submit \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "configuration_id": "YOUR_CONFIGURATION_ID",
    "url": "https://example.com/audio.mp3",
    "name": "My First Transcription"
  }'
```

```python Python
import requests

url = "https://api.heify.com/submit"
headers = {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY"
}
payload = {
    "configuration_id": "YOUR_CONFIGURATION_ID",
    "url": "https://example.com/audio.mp3",
    "name": "My First Transcription"
}

response = requests.post(url, headers=headers, json=payload)
data = response.json()
transcription_id = data["data"]["transcription_id"]
print(f"Transcription started: {transcription_id}")
```

</CodeGroup>

### Option B: Upload a Local File

For files on your computer, use the two-step upload process:

<Steps>
  <Step title="Request an upload URL">
    First, request a secure, pre-signed URL to upload your file.

    <CodeGroup>

```bash cURL
curl -X POST https://api.heify.com/request-upload-url \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "configuration_id": "YOUR_CONFIGURATION_ID",
    "name": "My First Transcription"
  }'
```

```python Python
import requests

url = "https://api.heify.com/request-upload-url"
headers = {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY"
}
payload = {
    "configuration_id": "YOUR_CONFIGURATION_ID",
    "name": "My First Transcription"
}

response = requests.post(url, headers=headers, json=payload)
data = response.json()

# Extract data needed for the next step
upload_data = data["data"]
upload_url = upload_data["upload_url"]
transcription_id = upload_data["transcription_id"]
meta_config_id = upload_data["configuration_id"]
meta_name = upload_data.get("name")
```

</CodeGroup>
    
    **Response:**
    
    The API will return the pre-signed `upload_url` and the metadata needed for the upload headers.
    
    ```json
    {
      "data": {
        "upload_url": "https://s3.amazonaws.com/...",
        "transcription_id": "1a2b3c4d...",
        "configuration_id": "a1b2c3d4...",
        "name": "My First Transcription"
      }
    }
    ```
    
    <Note>
      Save all the values from the `data` object for the next step.
    </Note>

  </Step>

  <Step title="Upload your audio file">
    Now, use the `upload_url` and metadata from the previous response to upload the file with a PUT request.

    <CodeGroup>

```bash cURL
# Use the 'upload_url', 'configuration_id', and 'name' from the Step 1 response
curl -X PUT --upload-file "path/to/audio.mp3" \
  -H "x-amz-meta-configuration_id: YOUR_CONFIGURATION_ID" \
  -H "x-amz-meta-name: My First Transcription" \
  "https://s3.amazonaws.com/..."
```

```python Python
import requests

# Use the variables extracted in the previous step
upload_headers = {
    'x-amz-meta-configuration_id': meta_config_id
}
if meta_name:
    upload_headers['x-amz-meta-name'] = meta_name

# Upload the file
with open("path/to/audio.mp3", "rb") as audio_file:
    response = requests.put(upload_url, data=audio_file, headers=upload_headers)

if response.status_code == 200:
    print(f"Upload successful! Transcription ID: {transcription_id}")
else:
    print(f"Upload failed: {response.text}")
```

</CodeGroup>
  </Step>
</Steps>

**Response (after a successful upload):**

A successful `PUT` request will return an HTTP `200 OK` status with an empty body. The transcription process has now started.

## Step 3: Check the Status

Monitor your transcription progress:

<CodeGroup>

```bash cURL
curl -X POST https://api.heify.com/details \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "transcription_id": "YOUR_TRANSCRIPTION_ID"
  }'
```

```python Python
import requests
import time

url = "https://api.heify.com/details"
headers = {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY"
}
payload = {
    # Use the transcription_id from either submission method
    "transcription_id": "YOUR_TRANSCRIPTION_ID"
}

# Poll until complete
while True:
    response = requests.post(url, headers=headers, json=payload)
    data = response.json()
    status = data["data"]["status"]
    
    print(f"Status: {status}")
    
    if status == "COMPLETED":
        print("Transcription complete!")
        print(data) # Print the full response data
        break
    elif status == "FAILED":
        print("Transcription failed:", data["data"]["error"])
        break
    
    time.sleep(5)  # Wait 5 seconds before checking again
```

</CodeGroup>
